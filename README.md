# Pequeno exemplo de uma aplicação Django/Mysql dockerizada.

## Para Rodar:

 OBS: É necessário que você tenha o Docker instalado.
 
 -----------------------------------------------------
 
 Para criar as imagens e subir os containers: 
```bash
 make docker-start
 ```
 
 Para rodar as migrações:
 ```bash
 make docker-db-migrate
 ```
 
 
 Pronto, só isso!
